Changelog

Dragon Budget project status

Transparent notes on what is available, what is planned, and what still needs work.

Current app status

As of May 3, 2026, the Dragon Budget public site and browser-only calculator are available. The fuller app is in Early Alpha.

Latest site updates

  • Created a no-build GitHub Pages site in the repository root.
  • Added the Can I Afford This calculator.
  • Added FAQ, comparison, pricing, privacy, terms, about, and guide pages.
  • Added Dragon Advisor preview, roadmap, design direction, and launch plan pages.
  • Added robots.txt, sitemap.xml, site.webmanifest, and structured data on key pages.

Planned features

  • Full Dragon Budget app planning.
  • More calculator controls and saved scenarios.
  • Transparent adaptive pricing for planned premium features.
  • More screenshots or product mockups as the app interface matures.

Known limitations

  • The calculator is manual and does not connect to bank accounts.
  • The result depends on the accuracy of the numbers entered.
  • Privacy and terms pages are starter pages that need review before production launch.

Feedback request

If you use the calculator, note whether the result felt clear, whether any inputs were missing, and whether the Yes/Maybe/No labels were useful. Early Alpha testers can also join the tester list for app feedback invitations.
