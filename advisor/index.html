Product direction

Dragon Advisor preview

Dragon Advisor is planned as a budgeting assistant that explains calculations using your local entries. It should not be a generic chatbot, and it should cite the bills, balances, subscriptions, and goals behind its answer.

Sample advisor answer

Why is my safe-to-spend low?

Entries cited

    Dragon Advisor should explain budget entries. It should not present output as financial advice or pretend to know data the user did not enter.

    Advisor rules for the app

    Cite the inputs

    Answers should name the bills, subscriptions, balances, goals, and planned items used in the calculation.

    Explain the math

    Safe-to-spend, cash-flow, and affordability answers should show the reasoning in plain language.

    Stay grounded

    If data is missing, the advisor should say what is missing and offer a next step instead of guessing.
