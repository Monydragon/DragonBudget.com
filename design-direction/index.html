Brand system

Design direction

The site now leans toward the mockup’s mature dark fintech mood: deep navy surfaces, blue and cyan highlights, emerald positive states, amber warnings, crisp cards, and a guardian-style dragon identity.

Mockup inspiration

The app preview suggests a premium budgeting cockpit: dashboard cards, cash-flow timeline, event budgeting, one top navigation menu, and a blue dragon presence that feels protective rather than childish.

Mascot direction

The mascot is a mature blue dragon guardian: sleek, intelligent, protective, and finance-aware. It should never read as babyish, goofy, or toy-like.

Dark mode first Light default on website Electric blue actions Green money states Cited advice Event budgeting

The public site does not pretend these app screens are already shipped. It uses the mockup as planning and brand direction for the app.

Design principles

Finance first

The interface should make money states, due dates, and risk clear before decorative elements.

Mature dragon identity

The dragon should feel like a calm guardian of the budget, not a toy mascot.

Transparent decisions

Every recommendation should explain the entries and calculations that created it.
