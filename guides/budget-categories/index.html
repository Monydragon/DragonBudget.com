Guide

Budget categories

Budget categories work best when they are specific enough to guide decisions but not so detailed that the budget becomes exhausting.

Choose useful categories

Start with essentials, flexible spending, subscriptions, savings, debt, and planned events. Add detail only where decisions need it.

Use thresholds

A warning at 70% or 80% spent can help before the category is gone.

Rollover carefully

Rollover can help with variable categories, but it should not hide overspending in essentials.
