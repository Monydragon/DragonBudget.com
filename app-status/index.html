Prompt 23

App status matrix

Dragon Budget currently offers a static website and browser-only calculator. The full app experience is planned, and future features are labeled clearly.

Available

Static calculator and public pages

The calculator, FAQ, guides, roadmap, advisor preview, design direction, and launch plan are available as static pages.

Planned

Manual-first budgeting MVP

Manual accounts, transaction entry, budgets, bills, subscriptions, goals, reports, cash-flow planning, and data export are planned app modules.

Planned

Dragon Advisor

The advisor is planned to answer questions from local budget entries and cite the entries it used.

Future

Bank sync and cloud sync

Bank sync and cloud sync are future architecture areas, not current site features. Privacy pages must be updated before either ships.

Not included yet

Accounts and payments

The static site does not create accounts, process payments, connect to banks, or store financial entries.
