Product direction

Event budgeting

Event budgeting helps answer whether a weekend trip, holiday, birthday, or one-time purchase fits without endangering bills and safe-to-spend.

Total budget

Start with the full event amount and split it into categories like transport, food, activities, and shopping.

Remaining amount

Track spent and remaining money during the event, not only after the weekend is over.

Affordability check

Compare the event budget against safe-to-spend and upcoming cash-flow pressure.

Current static tool

The event budget demo lets you model a simple event locally in the browser. The planned app would connect event budgets to cash flow and Advisor explanations.
